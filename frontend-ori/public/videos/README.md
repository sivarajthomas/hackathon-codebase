# Agent background videos

Drop optimized, muted, looping MP4 files here to enable per-agent cinematic
logistics backgrounds. File names must match each agent's `slug`:

- `invoice.mp4` — digital invoices, financial dashboards, billing analytics
- `rate.mp4` — pricing dashboards, shipping-cost visualization, analytics
- `shipment.mp4` — trucks, warehouses, cargo aircraft, containers, sorting
- `dispute.mp4` — customer support, claims review, document validation

Recommended encoding for performance:

- H.264 MP4, 1080p or 720p, ~2–4 Mbps, 8–15s seamless loop
- No audio track (files are muted anyway)

If a file is missing, the agent page automatically falls back to an animated
gradient + ambient glow, so the site always renders correctly.
